RPlay by bencall

A java port of shairport

Ever wanted to play music on one computer and hear it on a another one without any cable? RPlay is made for you!

Dependencies

- On Mac OS X: Nothing - On Windows: Apple Bonjour for Windows (http://support.apple.com/kb/DL999)

Install

Double click on the RPlay.jar file in the "binaries" folder should be enough.
